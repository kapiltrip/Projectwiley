# Overleaf Bundle

Upload this folder to Overleaf together with the Wiley template/class files.

Included:
- `wileyintegration.tex`
- `main.tex` (copy of the manuscript for direct Overleaf compilation)
- `references.bib`
- `images/` with 16 valid placeholder PDF figures matching the manuscript paths

These figure PDFs are placeholders for compilation only. Replace them with final figures before submission.
