Timing analysis algorithm using a neural network under PVT variations  
Yan Liu, Yan-Hua Ma*, Yu-Chun Chang**  
School of Microelectronics, Dalian University of Technology, Dalian, China

Keywords:  
Process-voltage-temperature variations result in an increase in logic gate delays, which further lead to timing  
violations. Because of the computational complexity of the operators such as max and sum in statistical static  
timing analysis, this paper proposes a timing analysis algorithm using neural network to evaluate circuit timing  
under the PVT variations. In the proposed algorithm, a neural network method is applied to model the mean or  
standard deviation of the gate delay. Then, a viability analysis method is introduced to recognize the true critical  
path. The simulation results on ISCAS89 benchmarks illustrate that compared with SPICE Monte Carlo simula-  
tion, the error in the mean calculated by the proposed algorithm is less than 2%, and the error in the standard  
deviation is less than 3.64%. In addition, for s298, s344, s832, s1238 and s13207, the viability analysis algorithm  
achieves 1.04X, 1.01X, 2.77X, 1.62X and 4.48X speedup over full path search, respectively.

With the size of transistors is continuously shrinking, the inevitable  
process variations that are the physical deviation of the parameters of  
transistors such as channel width and gate oxide thickness become more  
prominent [1]. The changes in physical parameters of transistors lead to  
the variations in circuit performance. Additionally, circuit performance  
is also affected by temperature and voltage [2]. The  
process-voltage-temperature variations are called the PVT effects.  
Because of these effects, the threshold voltage of transistors shifts,  
resulting in a change in logic gate delay, which eventually leads to the  
timing violation of the circuit [3]. Therefore, it is significant to evaluate  
the impact of PVT effects on circuit timing.  
Ref. [4] applied an artificial neural network to model the relationship  
between the gate delay and the process parameters. Considering the  
effect of temperature and voltage on circuit timing, a single delay model  
for the logic gate using multi-layer perceptron with a single hidden layer  
is employed to capture the process, voltage, temperature, load and input  
slew in Ref. [8]. The path-based SSTA method computes the delay of  
each path and estimates circuit timing with the maximum of all path  
delays [9]. However, with the increasing scale and complexity of the  
circuits, the number of paths that need to be analyzed rises accordingly,  
which further leads to expensive calculations.  
In addition, according to Hrapcenko’s analysis, there are either no  
false paths or a large number of false paths in the circuit [10]. Fig. 1 lists  
an example of the false path. To propagate a toy, m and z must be 1; x  
andb is 0. However, the output signaly is always 0 whenx andb are 0. In  
other words, path{a, c, z, y} is not sensitive to signals x andb. Due to the  
false paths, we can hardly detect the true critical path, which finally  
results in an inaccurate timing calculation [11].  
Studies on the impact of PVT effects mainly concern the statistical  
static timing analysis (SSTA) based methods, which calculate the dis-  
tribution function of circuit timing by propagating the hypothetical  
distribution function of logic gate delay [4,5]. The SSTA methods can be  
generally divided into block-based and path-based SSTA methods. The  
block-based SSTA method uses sum and max operators to propagate the  
delay distribution [6]. For simplifying the max operator, it is the  
Gaussian distribution that is often assumed to be the distribution of  
process parameters [3]. But practically, the Gaussian distribution is too  
ideal to calculate circuit timing accurately [7]. In Ref. [7], the kernel  
density estimation is used to build a look-up table which contains the  
performance under different operating conditions. However, the library

2. Timing analysis algorithm using a back-propagation neural  
network  
2.2. Viability analysis  
The overall framework of the proposed timing analysis algorithm  
consists of gate delay modeling and viability analysis method, as shown  
in Fig. 2. In the gate delay modeling stage, a neural network is applied to  
model the mean or standard deviation of the gate delay according to an  
appropriate number of SPICE Monte Carlo simulations. In the viability  
analysis stage, we first merge logic gates and their fan-outs into a graph;  
then all timing paths of the circuit can be found by a depth-first search  
algorithm. Next, based on the types of logic gates, the sensitivity con-  
ditions of the paths could be constructed. Finally, the true critical paths  
can be recognized according to the sensitivity conditions.  
According to Hrapcenko’s analysis, there may exist the false paths in  
circuit. Additionally, if the critical path in circuit is a false path, the  
estimated circuit timing is pessimistic. Therefore, it is very significant to  
identify the true critical path.  
To filter out the true critical path, we first collect all timing paths, the  
types of logic gates and the side input signals of the logic gates by the  
depth-first search algorithm. Subsequently, the sensitivity condition of  
the timing path is constructed. Thus, the true critical path can be  
recognized. An overview of the viability analysis is summarized in Al-  
gorithm 1.

4. Conclusion  
To address the PVT effects and the false paths problem, a timing  
analysis algorithm is proposed to evaluate circuit timing. We first  
construct a BPNN to model the mean or standard deviation of the logic  
gate delay. Then, the viability analysis is introduced to find the true  
critical path. The simulation results indicate that our algorithm can  
effectively predict the circuit timing when considering PVT effects and  
the false paths.
